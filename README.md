# chromasampler

[![License](https://img.shields.io/badge/license-GNU%20GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0.html)

Chromasampler is a command-line tool written in C++ that calculates the average
colour of an image and generates a user-specified number of shades of the
image's average colour in both RGB and hex formats.
