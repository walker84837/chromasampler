# ChromaSampler - Find average color and shades of an image

[![License](https://img.shields.io/badge/license-GNU%20GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0.html)

ChromaSampler is a simple C++ command-line utility using the OpenCV library, that calculates the average color of an image and provides 10 shades of that color in both RGB and hex formats. This program calculates the average color (RGB) of an image, converts the average color to hex format, and generates shades of the average color with adjustable intensity.
